Présentation du projet

Le Kap Hot aimerait bien avoir une application ludique, et qui permettrait aux étudiants de s'informer en même temps. C'est le but de ce projet. L'idée de base serait de faire un jeu à boire de type Picolo mais les règles ne sont pas encore établies donc vous aurez l'occasion de vous immerger d'autant plus dans le projet en proposant vos propres idées!

Ce que vous aurez l'occasion de faire

Dans ce projet, le principal élément est le développement mobile. Evidemment, vous n'aurez pas à faire l'entièreté du processus de création d'application mobile, vous vous concenterez sur la partie design et implémentation. Comme il n'y aura rien de fait pour le code, vous serez assez libre de vous organisez comme vous le voulez.

Ce que vous apprendrez

Pour les étudiants ne connaissant pas bien le développement android, vous pourrez approfondir vos connaissances dans ce domaine. Pour ceux qui sont déjà familier avec ça, il y a tout les autres aspects qui sont encore à explorer (base de données, interface utilisateur, etc.)
