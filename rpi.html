Présentation du projet

Le projet consiste en la réalisation d'une petite station météo individuelle et autonome sur base d'un nano-ordinateur RaspberryPi. L'unité sera chargée de la récolte, du traitement et de l'affichage des données.

Ce que vous aurez l'occasion de faire

Récolte des données

Cette partie du projet se concentre sur les interactions entre le RPi et les différents capteurs. Elle se fera majoritairement en Python. Nous prévoyons d'utiliser, par exemple, les capteurs suivants:

  • Température
  • Pression atmosphérique
  • Humidité
  • Luminosité

Bien sûr, nous restons ouverts à toutes propositions.

Traitement et stockage des données

Vous aurez aussi l'occasion de déterminer le format et la manière dont les données seront stockées. Il est envisageable d'utiliser une base de données classique (ex: sqlite) ou d'utiliser des formats fichiers tels que JSON ou YAML. De plus, vous pourrez définir une API afin de facilement récupérer les données pour les afficher.

Affichage des données

Afin de présenter les données, vous aurez à réaliser une webapp Python qui sera hébergée sur le RPi. De plus, vous aurez l'occasion d'utiliser d3.js dans le but de faire de superbes graphes. Pas de panique si vous n'avez jamais utilisé ces langages, nous ferons diverses présentations dans le but de vous y introduire et nous resterons à votre disposition pour toute question technique.

Ce que vous apprendrez

Vous apprendrez à manipuler le GPIO d'un RPi et de manière plus générale à utiliser et programmer sur cette plateforme. Vous utiliserez pour ce faire les langages Python et JavaScript. Au terme de cette activité, vous serez capables de créer et d'héberger vos propres webapp sur le nano-ordinateur Raspberry Pi.
